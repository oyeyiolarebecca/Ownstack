# OwnStack

OwnStack is a Bitcoin-native business infrastructure platform helping entrepreneurs create portable business identities, generate invoices, track payments, and receive Lightning payments.

## Features

* Nostr Login
* Business Profiles
* Invoice Generator
* Lightning Payment QR
* Payment Tracking Dashboard

## Tech Stack

* Next.js
* Tailwind CSS
* Supabase
* Nostr
* Bitcoin Lightning
* Bitnob Sandbox

## Mission

OwnStack helps entrepreneurs own their business identity and financial records independently using freedom-focused technologies.
